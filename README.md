<h3>课程介绍：</h3>
<p><a title="查看与 SLAM 相关的文章" target="_blank">SLAM</a>的定义</p>
<p>Localization：在给定地图的情况下，估计机器人的位姿。</p>
<p>SLAM：同时估计机器人的位姿和环境地图。</p>
<p>Mapping：在给定机器人位姿的情况下，估计环境地图。</p>
<p>SLAM解决的问题</p>
<p>机器人在环境中的位姿 导航过程中需要的环境地图</p>
